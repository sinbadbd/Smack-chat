# Smack

**SMACK** is a chat application. Which is based on RestApi.

**USE**
- **Alamofire**
- **Sockets** <a href="https://github.com/socketio/socket.io-client-swift">socketio</a>

- Alamofire with Authentication
- Register/Login
- Create user / Profile
- Add Channel
